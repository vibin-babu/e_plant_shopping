# 🌿 Green Haven Nursery (E-Plant Shopping App)

## 📌 Project Overview

Green Haven Nursery is a modern plant shopping web application that allows users to browse, select, and purchase a variety of plants online. The app provides a smooth shopping experience with cart management, product listings, and a clean user interface.

---

## 🚀 Features

* 🪴 Browse a variety of plants
* 🛒 Add items to cart
* ➕ Increase / ➖ Decrease product quantity
* ❌ Remove items from cart
* 💰 Dynamic total price calculation
* 🔄 Continue shopping option
* 🧾 Checkout interface (basic)

---

## 🛠️ Technologies Used

* Frontend: React.js
* State Management: Redux Toolkit
* Styling: CSS
* Routing: React Router

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/green-haven-nursery.git
```

2. Navigate to project folder:

```bash
cd green-haven-nursery
```

3. Install dependencies:

```bash
npm install
```

4. Run the project:

```bash
npm start
```
